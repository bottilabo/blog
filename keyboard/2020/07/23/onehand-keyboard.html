使い方

レイヤーキーと同時打鍵の組み合わせで様々な文字を入力できます。

LAYER key

LAY key Layer single tap
LAY0 chr layer Space
LAY1 Cursor Backspace
LAY2 Number Tab
LAY3 Fn keys  
LAY1+LAY2 LAY3  

with no LAYER key

ESC , s u t p
TAB a o e i c
SFT . x - h j
    LAY2 LAY1 LAY0 (LAY3)

longpress 長押し

0.2秒以上キーを長押しすることでシフトキーを押すのと同じ効果を得ることができます。

MOD key(SFT,CTL,ALT,GUI)

  • シングルタップでワンショットキーになり、次に押す１キーが影響を受けます。
  • ダブルタップでロックされ、更にもう一度MODキーを押すと解除されます。

日本語ローマ字モード

JP key(LAY1) で日本語ローマ字モードになります。 | from | to | |————–|—–| | c | k | | x | nn | | l(longpress) | wo | | q | ltu |

  • 子音とシフトキー（長押し）すると、子音のあとに y が付きます。
  • 母音とシフトキー（長押し）すると、母音のあとに nn が付きます。

ex. k(longpress) a(longpress) -> kyann
